# דוגמת weekly_summary — מאיפה מגיע כל אחוז שבועי

הדוגמה הזו מוכיחה, מול הפונקציות האמיתיות שב-`gather_review_input.py`, שני דברים:

1. **לכל אחוז שבועי יש שני עוגנים**: מחיר סגירה קודם מ-`market_history.json` + מחיר נוכחי מה-quote החי.
2. **אם ה-quote אינו מחיר סגירה אמין של סוף שבוע** (מחיר תוך-יומי או עסקה אחרונה שאינה של שישי של השבוע), **לא מוצג אחוז שבועי** לאותו סימבול.

להרצה מחדש: `python examples/weekly_history_demo.py`

## התרחיש

- `market_history.json` אחרי זריעת סגירות **10.7.2026** (שישי הקודם):
  `SPY 754.95, QQQ 725.51, META 669.21, MU 979.30`.
- הריצה השבועית עבור שבוע **13–17.7.2026** ‏(`monday=2026-07-13`, סגירת השבוע = שישי **2026-07-17**).
- ה-quotes שנתפסים בזמן הריצה (שבת 18.7), עם חותמת העסקה האחרונה (`ts`):

| סימבול | מחיר quote | עסקה אחרונה (`ts`) | עוגן קודם בהיסטוריה |
|---|---|---|---|
| SPY  | 761.80 | 2026-07-17 (סגירת שישי) | 754.95 @ 10.7 |
| META | 690.40 | 2026-07-17 (סגירת שישי) | 669.21 @ 10.7 |
| MU   | 968.10 | 2026-07-17 (סגירת שישי) | 979.30 @ 10.7 |
| TSLA | 402.10 | 2026-07-17 (סגירת שישי) | — (אין) |
| QQQ  | 731.00 | 2026-07-16 **תוך-יומי** | 725.51 @ 10.7 |

## הפלט (כפי שהמודל מקבל אותו)

```
WEEKLY PERFORMANCE (use THESE for weekly changes in the weekly summary, NOT the daily numbers):
  S&P 500 (SPY ETF): weekly +0.91% (from $754.95 to $761.80)
  $META: weekly +3.17% (from $669.21 to $690.40)
  $MU: weekly -1.14% (from $979.30 to $968.10)
```

### מקור כל אחוז (עוגן קודם + מחיר נוכחי)

```
S&P 500 (SPY ETF): prior $754.95 on 2026-07-10 (market_history.json) + current $761.80 settled 2026-07-17 (quote) = +0.91%
$META:             prior $669.21 on 2026-07-10 (market_history.json) + current $690.40 settled 2026-07-17 (quote) = +3.17%
$MU:               prior $979.30 on 2026-07-10 (market_history.json) + current $968.10 settled 2026-07-17 (quote) = -1.14%
```

### מה שלא הוצג לו אחוז שבועי (ולמה)

```
Nasdaq 100 (QQQ ETF): quote is a 2026-07-16 price, not a settled 2026-07-17 close   ← נכשל בעוגן 1 (סגירת סוף שבוע)
$TSLA:                no prior-week close in market_history.json                     ← נכשל בעוגן 2 (סגירה קודמת)
```

שים לב: ל-QQQ **יש** סגירה קודמת בהיסטוריה (725.51), ובכל זאת הוא נחסם, כי ה-quote שנתפס הוא מחיר יום חמישי תוך-יומי ולא סגירת שישי. זו בדיוק ההגנה שביקשת: מחיר שאינו סגירת סוף שבוע אמינה לא הופך לאחוז שבועי.

## איך זה נראה בסקירה עצמה

הבולטים משתמשים אך ורק בסימבולים שקיבלו אחוז שבועי אמיתי (SPY, META, MU). QQQ ו-TSLA לא יקבלו אחוז שבועי, ולכל היותר יתוארו איכותית או עם מהלך היום האחרון מסומן ככזה:

```json
{
  "summary": [
    "השבוע שהיה: השוק ננעל שבוע חיובי, ה-S&P 500 הוסיף כ-0.91% על פני השבוע.",
    "מטא ממשיכה להוביל: המניה עלתה כ-3.17% בשבוע על רקע ההשקעות בתשתיות ה-AI."
  ]
}
```
