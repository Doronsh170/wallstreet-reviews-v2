סקירות וול סטריט

הכנה ליום מסחר, עדכון ביניים, סיכום יומי וסיכום שבועי עם מבט לשבוע הבא — בעברית, על בסיס נתונים מאומתים

איך מפרסמים סקירה חדשה? (5 שלבים)
  1. בוחרים סוג סקירה ומריצים איסוף — Actions ‏← "1 - Gather Review Input" ← Run workflow ← בוחרים מצב (daily_prep / intraday_update / daily_summary / weekly_summary).
  2. מעתיקים את חומר הגלם לצ'אט — פותחים את raw_review_input.md ומדביקים את כל התוכן ב-Claude או ChatGPT (עם חיפוש אינטרנט). הצ'אט מחזיר JSON.
  3. מדביקים את ה-JSON — עורכים את review_output.json, מחליפים את התוכן בתשובת הצ'אט (מותר כולל ```json) ועושים Commit.
  4. הפרסום רץ אוטומטית — ה-Commit מפעיל את "2 - Publish Review" שמאמת ומעדכן את data.json. אם הוא אדום — פותחים את הלוג, מתקנים בצ'אט ומדביקים שוב.
  5. מרעננים את העמוד הזה — הסקירה החדשה מופיעה בטאב שלה.

כל התהליך עובד גם מהטלפון, דרך אתר GitHub בלבד.
